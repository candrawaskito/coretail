Introduction

Get started with Dash-ui, Responsive Website Template for building responsive, mobile-first sites, with Tailwind CSS and a template starter page.

Core libraries used

Here is a list of some core libraries that we use in Dashui:

  • TailwindCSS

    A utility-first CSS framework packed with classes.

  • Gulp

    We use Gulp as our build tool to bundle JavaScript & Css.

  • Bootstrap(Javascript)

    We included Bootstrap, but only javascript part for the ease of some component functionality.

Quick Start

Using this step you can run Dashui Tailwind Project.

  • npm install -g gulp-cli
  • npm install
  • run - gulp
  • build - gulp build
